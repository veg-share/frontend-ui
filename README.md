## Set-Up Instructions
1. Clone the repository
```git@github.com:christina-cook/rancid-tomatillos.git```

2. Install dependencies
```npm install```

3. Start the server
```npm start```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

5. Start the test suite
```npm run cypress```
